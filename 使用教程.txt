Vesper（夕语）使用教程 v0.5.0

一、安装

方式一：打包版（推荐，无需安装 Python）

1. 下载 `Vesper_v0.5.0.zip` 并解压到任意文件夹
2. 进入解压后的文件夹，双击 `Vesper.exe`
3. 首次启动会自动释放运行环境（约 30 秒），请耐心等待
4. 桌面右下角托盘会出现夕语图标，同时弹出聊天窗口

方式二：源码版（需要 Python 3.11+）

1. 下载 `Vesper_v0.5.0_source.zip` 并解压
2. 安装 Python 依赖：
   ```
   双击运行 安装依赖.bat
   ```
   或手动执行：
   ```
   pip install fastapi uvicorn pywebview chromadb sentence-transformers openai pystray Pillow vosk rank-bm25
   ```
3. 双击 `启动夕语.bat` 或执行：
   ```
   python launcher.py
   ```

系统要求

- Windows 10/11 64 位
- 至少 4GB 内存（推荐 8GB）
- 约 2GB 磁盘空间（含 AI 模型）
- 需要联网访问 AI 模型 API（DeepSeek / OpenAI 等）



二、首次配置（保姆级步骤）

第 1 步：选择角色

启动后会看到引导界面。选择一个喜欢的角色预设（如"弗洛洛"），点击下一步。

> 角色决定了 AI 的说话风格和性格。后续可以在「设置 → 角色人设」中查看和修改。

第 2 步：配置 AI 接口

这是最关键的一步。夕语本身不带 AI 模型，需要连接大语言模型 API。

推荐使用 DeepSeek（国产、便宜、中文好）：

1. 提供商选择「DeepSeek」
2. 填入 API Key（在 https://platform.deepseek.com 注册后获取，新用户送免费额度）
3. 点击「保存」，然后「测试连接」
4. 看到「连接成功」就可以点完成

> 其他支持的提供商：通义千问、Moonshot、智谱、OpenAI、Ollama（本地）
> 
> Ollama 用户无需 API Key，确保本地 Ollama 正在运行即可。

第 3 步：设置你的名字

在「设置 → 角色人设」中：
- AI 名字：给你的 AI 起个名字（默认"夕语"）
- 你的名字：告诉 AI 怎么称呼你



三、基本使用

聊天

在底部输入框输入文字，按 Enter 发送，Shift+Enter 换行。

- 消息会逐句弹出（可在设置中改为逐字、连续等模式）
- 顶部状态栏显示当前好感度和信任度
- 消息时间显示相对时间（"3分钟前"等）

快捷键

| 快捷键 | 功能 |
|--||
| Ctrl+1 | 聊天 |
| Ctrl+2 | 工具 |
| Ctrl+3 | 统计 |
| Ctrl+4 | 记忆 |
| Ctrl+5 | 设置 |
| End | 回到底部 |



四、工具使用

点击左侧工具栏的「工具」图标（或 Ctrl+2），可管理：

| 工具 | 说明 |
|||
| 待办 | 添加/完成/删除待办事项 |
| 笔记 | 写长笔记，支持拖拽调整大小 |
| 倒计时 | 设置目标日期（年/月/日下拉选择） |
| 提醒 | 设置定时提醒（年/月/日/时/分下拉选择） |
| 目标 | 设定长期目标，AI 会帮你追踪进度 |
| 日记 | 查看 AI 每天自动写的日记，或手动生成 |



五、设置说明

AI 接口配置
- 提供商：选择 LLM 服务商
- API 地址：通常保持默认即可
- 模型：可选不同的模型（点击"获取"按钮查看可用模型列表）
- 联网搜索：开启后 AI 可以搜索网络获取实时信息

角色人设
- 语气：冷静/活泼/温柔/毒舌/傲娇/自由
- 回复长度：从极简（5字）到不限
- 记忆回调：是否让 AI 主动提及过去的对话
- 颜文字：开启后 AI 会用颜文字表达情绪（如（笑）（叹气））
- 自定义提示词：高级用户可直接编辑系统提示词

分句模式
- 智能分句：自动按标点断句，短于12字的会自动合并
- 分隔符分句：AI 用分隔符控制断句
- 逐字显示：打字机效果
- 连续输出：一次性显示全部

主动频率
- 关闭：AI 不会主动说话
- 低：约 3 小时
- 中：约 40-120 分钟（根据你的回复率自动调整）
- 高：约 30 分钟
- 深夜 23:00-7:00 不打扰

主动风格
- 温暖关怀：像朋友一样关心近况
- 幽默调侃：带俏皮玩笑
- 简洁直接：15字以内说重点
- 自由发挥：AI 自行决定

定位
- IP 定位：根据网络 IP 获取大致城市（写入"手动城市"，不会覆盖 GPS 结果）
- GPS 精确定位：使用浏览器定位，精度更高（首次需授权，后续重启不再询问）
- 手动选择：省份/城市下拉选择，作为天气和问候的兜底
- 高德 API Key：可选，用于增强天气查询



六、常见问题

Q：AI 不回复？
A：检查「设置 → AI 接口配置」是否已配置 API Key 并测试连接成功。

Q：为什么 AI 回复很慢？
A：取决于 LLM 服务商的响应速度。DeepSeek 通常较快（2-5秒），本地 Ollama 取决于你的硬件。

Q：能换 AI 角色吗？
A：可以。在「设置 → 角色人设 → 自定义提示词」中直接编辑。也可以用「角色预设」保存和切换。

Q：数据存在哪里？
A：所有数据存储在程序目录下的 `data/` 文件夹中。`sakura.db` 是 SQLite 数据库，`chroma_db/` 是向量检索库。可以备份整个 `data/` 文件夹。

Q：如何迁移到新电脑？
A：旧电脑上：设置 → 数据管理 → 数据迁移 → 导出备份。新电脑上：相同位置 → 导入备份。导入后建议在「向量记忆引擎」中点击「重建向量索引」。

Q：忘了 PIN 码怎么办？
A：输错 3 次后会出现"忘记PIN?"链接，点击确认重置即可。聊天记录不会丢失。
