夕语（Vesper）—— 你的本地 AI 桌面伴侣
========================================

夕语是一个开源的本地 AI 桌面聊天应用。它运行在你的 Windows 电脑上，
无需云端服务（除 AI 模型 API 外），所有对话数据完全由你掌控。

版本: v0.5.0 (2026-05-25)
代码版本: 5.0.0

核心特性
--------
* 长期记忆  —— 记住你说过的话，不是每次重置的金鱼
* 人设锚定  —— 角色性格随互动自然演化，不会漂移
* 共情增强  —— 不只说"我理解你"，而是用不同策略接住情绪
* 主动关怀  —— 在你沉默时主动问候，融入天气和位置信息
* 工具集成  —— 待办、笔记、提醒、倒计时、日程、目标、日记
* 隐私优先  —— SQLite + ChromaDB 本地存储，零数据上传

技术栈
------
Python 3.11+ / FastAPI / Vue 3 / SQLite / ChromaDB
LLM 兼容 OpenAI API（DeepSeek / 通义千问 / OpenAI / Ollama）
桌面壳: pywebview (WebView2)
