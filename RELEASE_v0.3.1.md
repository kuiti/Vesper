## v0.3.1 更新内容（2026-05-22）

### 🎯 主动交互系统（全新）

AI 不再只是被动回复，会根据情境主动发起对话：

| 触发条件 | 说明 |
|----------|------|
| **A - 负面情绪关怀** | 检测到用户情绪负面时，主动关心安慰 |
| **B - 活跃时段问候** | 用户在活跃时间打开窗口时，主动打招呼 |
| **C - 持续低落检测** | 连续 3 天以上情绪低落后，AI 会主动询问 |
| **D - 提醒关联** | 临近提醒时间时，主动提及 |
| **E - 目标跟进** | 用户设定目标后，适时询问进展 |

主动频率受 AI 当前情绪状态（9 种情绪）、好感度/信任度、性格特征等多因素动态调节，不会频繁打扰。

### 🧠 AI 情绪自主演化引擎（全新）

AI 的情绪和性格会随时间自然变化，不再一成不变：

| 规则 | 说明 |
|------|------|
| **沉默衰减** | 用户 3+ 天未互动，好感度和信任度缓慢下降（最多 10 天） |
| **冷淡适应** | 用户持续情绪负面时，AI 进入自我保护状态，乐观度下降 |
| **习惯亲近** | 连续 28 天有互动，信任度缓慢积累 |
| **互动质量** | 正面互动多 → AI 变得更幽默开朗；用户活跃 → AI 表达欲增强 |
| **周日反思** | 每周日综合本周互动质量，微调全部性格特征 |

4 个性格维度：乐观度、表达欲、主动性、幽默感，各有高/中/低三档文字描述。

### 🔧 LLM 调用统一客户端

新建 `core/llm_client.py`，提供唯一入口 `call_llm()`：
- 统一管理 API Key、地址、模型
- 内置 JSON 模式（自动清洗 markdown fence → json.loads）
- 所有调用方（greeting、sentiment、split_sentences、goal_tracker、summary_engine、demand_analyzer、profile_builder）统一使用
- 消除了 8 个文件中的重复 HTTP 请求样板代码

### 📦 设置面板独立组件

SettingsPanel 从 App.vue 中提取为独立 Vue 组件：
- 自包含的 API 调用和状态管理
- 通过 props/events 与主应用通信
- 更清晰的代码结构，便于维护

### ✂️ 分句系统增强

新增分隔符分句模式：
- LLM 在生成回复时插入 `<<>>` 标记句子边界
- 前端实时检测分隔符，逐句弹出显示
- LLM 忽略分隔符时自动回退 DFA 状态机分句，确保不掉链

### 💭 记忆主动提及

AI 现在会在话题相关时，以约 5% 概率自然穿插"我记得你之前说过..."：
- 仅在话题明显相关时触发
- 每次对话只提一次，不重复
- 语气自然不做作，不硬扯

### 🐛 问题修复

| 问题 | 修复 |
|------|------|
| 多实例端口冲突 | 改为 5 个固定端口（8060-8064）+ HTTP 健康检查 |
| 滚动不到底部 | `scroll-behavior: smooth` 与 `loadMoreHistory` 冲突，加 `_blockLoadMore` 标志解决 |
| 亮色主题变量失效 | `:root[data-theme="light"]` 从 scoped style 移至全局 style |
| 聊天管理面板重复样式 | 合并两处 `.date-range` CSS 定义 |
| 倒计时重复计算 | 改为 computed 缓存天数 |
| 设置面板 API 路径错误 | 修正 location、avatar、upload 端点路径 |
| STT 静默失败 | 无 ffmpeg 时明确报错而非返回空 |
| test_search 阻塞事件循环 | DDGS 调用包装到 asyncio.to_thread |
| 端口管道句柄泄漏 | 添加 try/finally 正确关闭 |
| DB schema 不一致 | `_ensure_db()` 与 CRUD 函数列名对齐 |

## 安装方式

### 方式一：EXE 一键运行（推荐）

1. 从 [Releases](../../releases) 下载 `Vesper_v0.3.1_windows_x64.zip`
2. 解压到任意目录
3. 双击 `Vesper.exe` 运行
4. 首次启动自动创建 `data/` 和 `config.json`
5. 点击右上角齿轮 → API 设置 → 填入 API Key
6. 开始聊天

### 方式二：源码运行

```bash
git clone https://github.com/kuiti/Vesper-XiYu.git
cd Vesper-XiYu
python -m venv venv
pip install -r requirements.txt
python launcher.py
```

## 配置说明

### API 设置

1. 点击右上角齿轮 → API 标签
2. 填入 API Key（DeepSeek / OpenAI 兼容）
3. 默认 API 地址：`https://api.deepseek.com/v1`
4. 默认模型：`deepseek-chat`

### 主题切换

设置 → 外观 → 主题：
- **暗色**：深色背景，护眼
- **亮色**：浅色背景，清爽
- **樱花粉**：粉色系，温暖
- **夕语星光**：青紫渐变，神秘优雅

### 主动交互频率

设置 → 聊天 → 主动频率：低 / 中 / 高

### 分句模式

设置 → 聊天 → 分句方式：
- **智能分句**：DFA 状态机自动识别句子边界
- **分隔符分句**：AI 插入分隔符 + 前端回退兜底
- **逐字显示**：打字机效果逐字显示
- **连续输出**：整段输出，不拆分
