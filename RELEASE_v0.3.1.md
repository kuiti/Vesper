## v0.3.1 更新（2026-05-22）

### 新增
- **主动交互系统**：5 条件智能触发（负面情绪关怀、活跃时段问候、持续低落检测、提醒关联、目标跟进）
- **AI 情绪自主演化引擎**：沉默衰减、冷淡适应、习惯亲近、周日反思，AI 性格随时间自然变化
- **LLM 调用统一客户端**：消除 8 个文件中的重复 HTTP 样板代码
- **设置面板独立组件**：从 App.vue 提取，更清晰的代码结构
- **分隔符分句模式**：LLM 输出时自然分句，前端回退 DFA 兜底
- **记忆主动提及**：当话题相关时，低概率自然穿插"我记得你之前说过..."

### 修复
- 端口冲突 → 5 固定端口 + HTTP 健康检查
- 滚动到底部问题（smooth 动画与 loadMore 冲突）
- CSS `:root` 主题变量作用域问题
- 聊天管理面板重复样式
- 倒计时重复计算性能优化

### 移除
- 语音功能（STT/TTS，将在未来版本重新实现）

## 安装方式

### 方式一：EXE 一键运行（推荐）

1. 下载 `Vesper.exe` 压缩包
2. 解压，双击 `Vesper.exe`
3. 首次启动会自动创建数据目录
4. 在设置中配置 API Key
5. 开始聊天

### 方式二：源码运行

```bash
python -m venv venv
pip install -r requirements.txt
python launcher.py
```
